Resource Rage
Roles Tank, DPS
Specs Battlecraft (DPS) · Bloodrush (DPS) · Ironguard (Tank)

Specializations

  • Battlecraft — DPS. Signature ability: Maiming Strike (learned at level 5).
  • Bloodrush — DPS. Signature ability: Bloodletting (learned at level 5).
  • Ironguard — Tank. Signature ability: Shieldcrack (learned at level 5).
